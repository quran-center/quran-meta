    • QuranMeta

    Type Alias QuranMeta

    Represents the complete metadata structure for Quranic information. Contains counts for all structural divisions of the Quran.

    Remarks

    Different riwayas may have different values for these properties. Most notably, Qalun has 6214 ayahs while Hafs has 6236 ayahs.

    Example

    import { meta, riwayaMeta } from 'quran-meta'

    // Hafs metadata (default)
    console.log(meta.numAyahs)  // 6236

    // Qalun metadata
    console.log(riwayaMeta.Qalun.numAyahs)  // 6214
    type QuranMeta = {
        numAyahs: number;
        numHizbs: number;
        numJuzs: number;
        numManzils: number;
        numPages: number;
        numRubAlHizbs: number;
        numRubsInJuz: number;
        numRukus: number;
        numSajdas: number;
        numSurahs: number;
        numThumunAlHizbs: number;
        riwayaName: RiwayaName;
    }
